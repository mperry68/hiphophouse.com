# Images Folder

Place all your photos and images in this folder.

## Recommended Structure

You can organize images by category:
- `events/` - Event photos
- `gallery/` - Gallery images
- `logos/` - Logo files
- `banners/` - Banner images

## Image Formats

Recommended formats:
- **Photos**: JPG, JPEG (for photos with many colors)
- **Graphics**: PNG (for images with transparency)
- **Web**: WebP (for optimized web delivery)

## Usage in HTML

Reference images like this:
```html
<img src="images/your-image.jpg" alt="Description">
```
